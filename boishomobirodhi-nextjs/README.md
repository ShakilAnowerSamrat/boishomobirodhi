# বৈষম্যবিরোধী ছাত্র আন্দোলন - Next.js Website

Modern Next.js archival website for বৈষম্যবিরোধী ছাত্র আন্দোলন 2024.

## Tech Stack

- Next.js 15 + React 18 + TypeScript 5
- Tailwind CSS 4 + Framer Motion
- Radix UI + Lucide Icons

## Setup Instructions

### 1. Copy Logo File

**IMPORTANT**: Manually copy the logo file:

```
From: Image\logo\logo.png.png
To: boishomobirodhi-nextjs\public\images\logo\logo.png.png
```

Create the `public\images\logo` folder first if it doesn't exist.

### 2. Install Dependencies

Open terminal in the `boishomobirodhi-nextjs` folder and run:

```bash
npm install
```

### 3. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### 4. Build for Production

```bash
npm run build
```

Static files will be in the `out` folder, ready for GitHub Pages.
